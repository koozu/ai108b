# 語言處理
- 概念
    - 將語言分成詞彙、語句、文章來分析
    ```
    S=NP+VP
    NP=D+N
    VP=V+NP
    N=名詞詞彙
    V=動詞詞彙
    D=定詞
    ```
## 格狀語法
- 概念
    - 以角色與動作為核心，將物品分類，把句子對應到的詞彙放進欄位，此方式可得知句子的主要對象
## 統計式語言處理
- 概念
    - 輸入許多句子，判斷各字詞的左右字詞的多樣性，當左右多樣性都很高時，即為詞彙
