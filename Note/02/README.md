# 爬山演算法
- 想法
    - 往高處爬，若旁邊比較高，則往上爬
    
        ```
        p=初始點
        while p的鄰居p'比p更高:
            p=p'
        ```
- 缺點
    - 只能搜尋到區域最佳解，找不到全域最佳解
